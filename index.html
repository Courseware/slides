1st Let me introduce you BuddyPress

  • It's Free Software! Free as in speech!
  • Built on WordPress, using bbPress
  • As simple as installing Hello Dolly, needs as minimum requirements as WordPress has!
  • Easily customizable, very hackable!
  • Awesome community, really awesome!
  • Documentation doesn't matter how bad is, we offer support over IRC! ☺
  • Shares the same codebase, plugins and developers.
  • Started by Andy Peatling, with a bunch of plugins, resulted into this.
  • Targets: ning.com ex-users, intranets, niche social networks

Technical Details

  • Everything is WordPress
  • 0 SQL lines of code, post types for the win
  • Full usage of BuddyPress API
  • ~5000 lines of code
  • Hooks and actions for your plugins
  • WordPress as WebApps Dev Platform
  • Custom Post Types
  • A group → a class, fully integrated UI, Activity Streams Api, Notifications/Emails, Forums
  • WVC (WordPress–View–Controller) hooks and filters is all you need!
