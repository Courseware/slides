BuddyPress — Social Network In a Box

  • O platformă pentru crearea unei rețele socială cu cod liber
  • Construită pe platforma WordPress, folosind bbPress
  • Simplu de instalat, folosit, cu cerințe minime!
  • Personalizabilă, localizabilă
  • O comunitate enormă
  • Documentație proastă, cod accesibil! (:
  • Moștenește compatibilitatea cu o bună parte din modulele existente WordPress și bbPress
  • Început de Andy Peatling, de la început au fost câteva module care ulterior au fost integrate.
  • Target-ul actual e ex-utilizatorii ning.com și soluțiile pentru intranet-uri

Detalii Tehnice

  • Totul e WordPress
  • 0 linii de cod SQL
  • Integrare la toate nivelele cu BuddyPress
  • ~5000 de linii de cod
  • hackable & extensibil
  • WordPress API aka WebApps Dev Platform
  • Custom Post Types
  • Grup == Clasă, UI integrat, Activity Streams, Notificări, Forum
  • WVC (WordPress–View–Controller), hook-uri și filtre
